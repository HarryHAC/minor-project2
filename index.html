ALS Communication System

A Multi-Modal Assistive Communication System for ALS Patients

Combining Blink Detection, Air-Writing, Eye-Gaze, and EEG Control with Speech Output and Smart Home Integration

Project Overview

An innovative assistive technology solution for ALS patients

Blink Detection

Advanced computer vision algorithms to detect intentional blinks for communication and control.

Air-Writing Recognition

Real-time hand tracking and OCR to convert air gestures into text using MediaPipe and Tesseract.js.

EEG Control

Brain-computer interface integration for neural signal-based control of the system.

Smart Home Integration

Seamless control of IoT devices and smart home systems for enhanced independence.
